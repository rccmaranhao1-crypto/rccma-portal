# Design System — RCC MA (cores oficiais)

**Somente estas cores podem ser usadas no projeto.**

- Verde principal: `#006036`
- Verde secundário: `#009049`
- Verde escuro: `#004058`
- Amarelo: `#fdc800`
- Azul claro apoio: `#b8d0dc`
- Branco: `#ffffff`

Regras:
- Não introduzir outras cores.
- Gradiente padrão: `#006036 → #fdc800`
- Linhas curvas suaves (SVG), aparência institucional.
